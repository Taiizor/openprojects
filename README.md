## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/furkanczay/openprojects.git

# Install dependencies
cd openprojects
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
